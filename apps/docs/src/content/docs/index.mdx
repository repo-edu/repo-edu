## What repo-edu provides

- Electron desktop shell (`apps/desktop`) for interactive workflows.
- TypeScript CLI (`apps/cli`) with command name `redu`.
- Browser-safe docs/demo app (`apps/docs`) that mounts the real shared React app with mock host adapters.
- Shared domain and workflow packages used by all delivery surfaces.

## Typical CLI flow

<span class="line"><span class="prompt">$</span> redu profile list</span>
