# Output Formats

Reference for roster export formats generated by repo-edu.

## YAML (Students File)

YAML roster export format. Compatible with RepoBee.

### Structure

```yaml
# students.yaml
- name: team-alpha
  members:
    - alice@university.edu
    - bob@university.edu
- name: team-beta
  members:
    - charlie@university.edu
- name: solo-student
  members:
    - diana@university.edu
```

### Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Team/group name (used in repo naming) |
| `members` | array | List of member identifiers |

### Member Identifier Formats

The format depends on the **Member Option** setting:

| Option | Format | Example |
|--------|--------|---------|
| Email Only | `email` | `alice@university.edu` |
| Email and Git ID | `email` or `gitid` | `alice` (extracted from email) |
| Git ID Only | `gitid` | `alice` |

### Solo Students

Students not in a group are output as single-member teams:

```yaml
- name: alice-doe        # Derived from student name
  members:
    - alice@university.edu
```

## CSV

Spreadsheet-compatible format for analysis or import into other tools.

### Structure

```csv
group,member,email,name
team-alpha,alice@university.edu,alice@university.edu,Alice Doe
team-alpha,bob@university.edu,bob@university.edu,Bob Smith
team-beta,charlie@university.edu,charlie@university.edu,Charlie Brown
```

### Columns

| Column | Description |
|--------|-------------|
| `group` | Team/group name |
| `member` | Member identifier (per member option) |
| `email` | Student email address |
| `name` | Student full name |

### Options

The output varies based on settings:

| Setting | Effect |
|---------|--------|
| Include Group | Adds `group` column |
| Include Member | Adds `member` column |
| Include Initials | Appends initials to member ID |

## XLSX (Excel)

Excel format with the same structure as CSV, plus:

- Header row formatting
- Column auto-width
- Data validation (where applicable)

## File Naming

Default filenames (configurable in settings):

| Format | Default Filename |
|--------|------------------|
| YAML | `students.yaml` |
| CSV | `student-info.csv` |
| XLSX | `student-info.xlsx` |
