## What to Expect

The demo provides a fully functional simulation of the repo-edu desktop app running directly in
your browser. All backend interactions are mocked, so you can safely explore every feature without
connecting to real services.

## Demo Scenarios

The simulation includes pre-configured scenarios to showcase different workflows:

- **New Course Setup** - Start from scratch: configure LMS connection, import roster, create repos
- **Mid-Semester** - Work with an existing roster: add a new assignment, manage groups
- **Bulk Operations** - See how repo-edu handles large classes with 200+ students
