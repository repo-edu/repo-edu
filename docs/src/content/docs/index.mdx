## What is repo-edu?

**repo-edu** is a tool for educators who manage programming courses. It bridges your Learning
Management System with Git hosting platforms, eliminating the tedious manual work of setting up
student repositories.

<div class="feature-grid not-content">
  <div class="feature-card">
    <h3>LMS Integration</h3>
    <p>Import student rosters and group assignments directly from Canvas or Moodle. No manual data entry required.</p>
  </div>
  <div class="feature-card">
    <h3>Batch Repository Creation</h3>
    <p>Create dozens or hundreds of repositories in seconds using your templates on GitHub, GitLab, or Gitea.</p>
  </div>
  <div class="feature-card">
    <h3>Flexible Workflow</h3>
    <p>Use the desktop app for interactive work or the CLI for automation and scripting.</p>
  </div>
</div>
