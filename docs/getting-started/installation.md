# Installation

## Download

::: warning Work in Progress
Download links will be available when releases are published.
:::

RepoManage is available for:

- **macOS** (Apple Silicon and Intel)
- **Windows** (64-bit)
- **Linux** (AppImage)

## System Requirements

- macOS 11+, Windows 10+, or Linux with GTK3
- Internet connection for LMS integration

## First Launch

On first launch, RepoManage will create a configuration directory:

| Platform | Location |
|----------|----------|
| macOS    | `~/Library/Application Support/repo-manage/` |
| Windows  | `%APPDATA%\repo-manage\` |
| Linux    | `~/.config/repo-manage/` |
